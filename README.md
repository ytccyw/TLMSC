This is an experimental demo for TLMSC. The demo involves PCA, UMAP, Isomap, LLE, so the requirements are as follows.
## Requirements
- Python
- numpy
- scipy
- scikit-learn
- numba
- tqdm
- pynndescent
## Running
The code is run by Matlab, so set the Python location for the environment using the following command on the Matlab command line.

pyenv('Version', 'D:\envs\python.exe')

'D:\envs\python.exe' is an example.
